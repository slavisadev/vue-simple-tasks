# vue-simple-tasks

Simple app using Vue with Bootstrap 4.

####Includes Vuex showcase pattern and a simple router example.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```
